- Manually pushed [releases](https://github.com/simbleau/resume/releases) also trigger workflows to upload artifacts.
  - Only tagged releases upload a cover letter artifact.

# üîß Building

## Commands
- **Build the r√©sum√©**:
  ```bash

  ```
- **Build the cirriculum vitae**:
  ```bash

  ```
